# 学习通全能助手 - 乱选速刷版

## 📦 项目说明

这是基于 [xuexitong_addon](https://github.com/dongfang281127/xuexitong_addon) 原版的修改版，主要改动：

- ✅ **删除抓取模块**（scraper.js + xlsx库）
- ✅ **删除题库上传功能**（UI简化）
- ✅ **重写自动答题为乱选模式**（每题随机选1个选项）
- ✅ **增加自动切换**：刷课完成后自动切换到答题模式
- ✅ **体积缩小**：从 ~950KB 降至 ~30KB

---

## 🎯 核心功能

### 1. 智能刷课
- 自动识别视频/文档任务
- 2.0x 倍速静音播放
- 防暂停机制
- 自动翻到下一节

### 2. 无脑乱选
- **不需要题库**，每题随机选1个选项
- 自动跳过已批阅题目
- 自动提交（如有提交按钮）
- 自动翻页

### 3. 自动模式切换
- 刷课完成 → 检测到未答题 → 自动开启乱选模式
- 无需手动切换标签

## 🚀 安装使用

### 1. 加载插件（Chrome/Edge）

1. 打开浏览器扩展管理页
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
2. 开启 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择 `xuexitong_addon_fast` 文件夹
5. 插件安装完成

### 2. 使用流程

#### 模式A：只刷课（不答题）
1. 进入学习通课程页面
2. 点击插件面板 → "📺 刷课" 标签
3. 点击 **"▶ 一键完成页面任务"**
4. 视频/文档自动播放 → 完成后自动翻页

#### 模式B：只答题（乱选）
1. 进入学习通测验页面
2. 点击插件面板 → "⚡ 答题" 标签
3. 点击 **"⚡ 开始乱选"**
4. 每题随机选1个选项 → 提交 → 翻页

#### 模式C：刷课完自动答题（推荐）
1. 进入课程页面（视频+测验混合）
2. 点击 **"📺 刷课" → "▶ 一键完成页面任务"**
3. 视频播完自动翻页
4. 如果新页面有题目 → **自动切换到答题模式** → 乱选
5. 答题完成 → 再翻页 → 循环

## 🔧 技术细节

### 乱选逻辑
```javascript
// solver.js 核心算法
const type = detectType(qDiv);  // 识别题型
let count = 1;                  // 统选1个
const options = getOptions(qDiv); // 获取所有选项
const selected = shuffle(options).slice(0, count); // 随机选
selected.forEach(opt => opt.div.click()); // 点击
```

**题型识别优先级**：
1. 单选框 → 选1个
2. 多选题 → 也选1个（你指定）
3. 判断题 → 选1个
4. 未知题型 → 选1个（可扩展为随机1-3个）

### 自动切换检测
```javascript
// 每2秒检查一次
if (IS_LEARN_MODE && !IS_LEARN_RUNNING) {
    const unanswered = 未批阅题数量;
    if (unanswered > 0) {
        IS_LEARN_MODE = false;
        IS_ANSWERING = true;  // 自动开启答题
    }
}
```

## ⚠️ 注意事项

1. **分数不重要**：乱选模式正确率随机，只求完成不计分
2. **已批阅题目跳过**：如果题目已经显示✅/❌，不会重新选择
3. **题型识别可能不准**：如果页面class不同，可能多选误判为单选
4. **提交按钮**：如果题目需要"提交"才批阅，插件会尝试找提交按钮点击
5. **纯前端运行**：数据只存在浏览器本地，不会上传服务器
