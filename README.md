> **⚠️ 免责声明**：本项目仅供同学间技术交流与学习参考，严禁用于任何形式的作弊或违反学校规定的行为。使用者需自行承担所有责任。
>
> **⚠️ 项目状态**：这是一个非常拙劣的初版，包含大量已知和未知的 Bug，功能极其不稳定，仅供技术交流使用，请勿在正式环境中依赖此工具。

这是一个基于浏览器扩展的自动化脚本，旨在通过技术手段分析网页结构，尝试自动完成在线课程平台的视频观看和章节测验任务。

## 🚀 主要功能

1.  **自动刷课**：
    *   自动识别视频和文档任务。
    *   支持倍速静音播放。
    *   防暂停机制，确保视频连续播放。
    *   播放完成后自动翻页。

2.  **自动答题 (乱选模式)**：
    *   无题库模式：每题随机选择选项（单选选1个，多选选1-2个）。
    *   自动识别题目类型（单选、多选、判断）。
    *   自动检测并跳过已批阅题目。
    *   自动点击提交按钮。

3.  **智能模式切换**：
    *   视频播放完成后，自动检测是否进入章节测验页面。
    *   检测到测验时自动切换到答题模式。
    *   答题完成后自动跳转到下一节。

## 📦 安装指南

### 1. 下载代码
点击 GitHub 页面右上角的 "Code" -> "Download ZIP"，解压到本地文件夹。

### 2. 加载扩展 (Microsoft Edge)
1.  打开浏览器扩展管理页：
    *   Edge: `edge://extensions/`
2.  开启 **开发者模式** (左下角开关)。
3.  点击 **加载已解压的扩展程序**。
4.  选择解压后的文件夹。
5.  插件安装完成，浏览器右上角会出现插件图标。

> **注意**：虽然主要针对 Edge 开发，但基于 Chromium 的浏览器（如 Chrome）通常也能运行，但不保证兼容性。

## 🛠️ 使用方法

### 模式 A：只刷课（不答题）
1.  进入课程页面。
2.  点击插件面板 → "📺 刷课" 标签。
3.  点击 **"▶ 一键完成页面任务"**。
4.  视频/文档将自动播放 → 完成后自动翻页。

### 模式 B：只答题（乱选）
1.  进入测验页面。
2.  点击插件面板 → "⚡ 答题" 标签。
3.  点击 **"⚡ 开始乱选"**。
4.  每题随机选1个选项 → 提交 → 翻页。

### 模式 C：全自动流程（推荐）
1.  进入课程页面（视频+测验混合）。
2.  点击 **"📺 刷课" → "▶ 一键完成页面任务"**。
3.  视频播完自动翻页。
4.  如果新页面有题目 → **自动切换到答题模式** → 乱选。
5.  答题完成 → 再翻页 → 循环。
tip：有时候刷完题目之后会自动跳转下一个视频但不会自动播放，则需再点一次一键完成

## ⚙️ 配置说明

*   **视频速度**：在刷课面板的下拉菜单选择倍速（1.0x, 1.25x, 1.5x, 2.0x）。
*   **清空数据**：点击 "🗑️ 清空所有缓存数据" 可重置插件状态。

## 🔧 技术细节

*   **开发环境**：Edge Extension Manifest V3
*   **运行方式**：纯前端脚本，数据仅存储在本地浏览器中。
*   **答题算法**：基于随机选择的乱选模式，不依赖题库，保证完成率但不保证正确率。

## ⚠️ 注意事项

1.  **仅供交流**：本插件仅用于技术学习和交流，请勿用于违反学校规定的用途。
2.  **分数为零**：由于找不到题库，为了保证完成任务点进入下一节，本插件会提交空选，导致测验分数为0，介意分数者勿用！！！

4.  **页面适配**：目标平台页面结构可能会更新，如遇失效请检查选择器是否匹配。
5.  **免责声明**：使用本插件产生的任何后果由使用者自行承担。
6.  **版本拙劣**：这是一个非常简陋的版本，包含大量 Bug，代码质量低，功能不稳定。
